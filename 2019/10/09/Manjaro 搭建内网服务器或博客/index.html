使用 wordpress

还是建议用 wordpress ，更加成熟，
安装方式在 wordpress 官网上把包下下来解压到 /srv/http/ ，
回了正常运行，需要改变 /srv/http 的权限，让 wordpress 能够对其做出修改。
这里不赘述，嫌麻烦可以 chmod 777 -R /srv/http 。
